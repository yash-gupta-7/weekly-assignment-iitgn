## 🚀 Repository Highlights

1. **Modular Engineering**: Assignments follow a pattern of `data_generator.py` → `src/` modules → `notebooks/` analysis for reproducible pipelines.
2. **From-Scratch Mathematical Rigor**: Implementations like TF-IDF engines and Neural Networks are built in pure NumPy/SciPy before library comparisons.
3. **Business & Clinical Focus**: Models are evaluated using domain-specific costs (e.g., Clinical Readmission costs, Sentiment Misclassification costs).
